The Naming Game

This project focuses on a self-organized communication system: The naming game. In this system, each agent chooses a particular state, or opinion, by means of a memory-based negotiation process, during which a variable number of states is collected and kept in memory. The influence of agents' diversity on this communities is mainly discussed.
